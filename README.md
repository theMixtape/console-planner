# console-planner

Мінімалістичний консольний планувальник подій на Python. Дозволяє керувати завданнями, зустрічами та нотатками безпосередньо через термінал.

---

### ⚙️ Функціонал
* **Додавання подій** ➜ створення запису з назвою, датою, часом та описом.
* **Перегляд списку** ➜ послідовне виведення всіх запланованих справ.
* **Видалення записів** ➜ видалення подій за унікальним ID з підтвердженням.
* **Автозбереження** ➜ усі дані автоматично записуються у файл `planner.json`.

---

### 🛠️ Швидкий запуск

1. Переконайтеся, що у вас встановлений **Python 3**.
2. Відкрийте термінал у папці з проєктом та запустіть:
   ```bash
   python planner.py
